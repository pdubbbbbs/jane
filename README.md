# Jane

Dark standalone desktop app for Agent S. Cyberpunk theme, no browser chrome.

## Install (macOS)

```bash
git clone https://github.com/pdubbbbbs/jane.git
cd jane
npm install
npm start
```

## Build macOS app

```bash
npm run build-mac
```

The `.app` will be in `dist/mac/`. Drag to Applications.

## Colors

Edit the CSS variables in `main.js` to customize the palette.
